# katieheinemann.space

Personal website for Katie Heinemann — Aerospace Engineer, Software Engineer, MPH.

Live at [katieheinemann.space](https://www.katieheinemann.space) · hosted on GitHub Pages.

## Structure

```
index.html        # Home
portfolio.html    # Project portfolio
resume.html       # Work history, education, certifications
giving-back.html  # Pro bono / nonprofit engineering offer
contact.html      # Social links
css/style.css     # Styles
img/              # Images (katie-nbl.webp, etc.)
```

## Deployment

Pushing to `main` automatically deploys via GitHub Actions (`.github/workflows/deploy.yml`).

## Local development

Open any `.html` file directly in a browser — no build step required.
